# Plastic Bottle – AutoCAD 3D Design

This repository presents a 3D CAD model of a plastic bottle designed using AutoCAD.

## 🧴 About the Project
The model represents a typical PET bottle used in beverage packaging. It includes key features such as the neck, body, and base geometry.

## 🛠️ Tools Used
- AutoCAD

## 📌 Design Features
- Cylindrical body for uniform stress distribution
- Threaded neck for cap fitting
- Petaloid base design for stability
- Symmetrical 3D modeling

## 🏭 Manufacturing Consideration
The design is based on the **blow molding process**, commonly used for plastic bottles.  
The base shape helps in pressure distribution, and the neck ensures proper sealing.
The design also considers material efficiency and structural stability under internal pressure.
